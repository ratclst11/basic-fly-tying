# Basic Fly Tying Project

This repository was created to practice using Git and GitHub while building resources related to fly tying. It includes documentation, images, and notes that support my work on basic fly tying techniques, patterns, and materials.

## Purpose
The purpose of this repository is to learn:
- How to create and manage Git repositories
- How to track changes using Git
- How to upload and organize files in GitHub


## Functionality
This repository contains:
- A README describing the project
- A Word document discussing Git best practices
- A fly‑tying related image


## Potential Applications
This workflow can support:
- Organizing fly‑tying patterns
- Storing step‑by‑step tying instructions
- Managing photos of flies, tools, and materials
